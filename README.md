🧩 TEAcolhe - Autism Support and Awareness Platform
Project Status SEO TEAcolhe is a Single Page Application (SPA) designed to provide accessible, welcoming, and scientifically grounded information about Autism Spectrum Disorder (ASD). The focus of this project goes beyond visuals: it is an application optimized for search engines (Technical SEO), ensuring that information reaches those who need it, with uncompromised accessibility and performance.

🚀 Technical & Engineering Highlights
1. Advanced Technical SEO (Search Engine Optimization)
Unlike standard websites, TEAcolhe implements advanced indexing strategies:

JSON-LD (Structured Data): Implementation of structured data following the Schema.org standard, allowing Google to understand the semantic context of the page (Rich Snippets).
Open Graph Protocol: Optimized meta tags to guarantee perfect previews when sharing the link on WhatsApp, LinkedIn, and Facebook.
Meta Viewport & Canonical: Precise configurations to prevent duplicate content and ensure perfect mobile rendering.
2. UI/UX & Design System
Glassmorphism: Modern use of transparency and blur (backdrop-filter) to create a light and welcoming aesthetic.
Complex Responsive Design: The layout fluidly adapts from mobile phones to ultrawide monitors, fixing common "background cut-off" bugs on large screens.
Accessibility: Color contrast designed for comfortable reading and proper HTML semantic structure.
🛠️ Technologies Used
Core: Semantic HTML5, Modern CSS3 (Flexbox/Grid/Variables), Vanilla JavaScript.
SEO: JSON-LD, Strategic Meta Tags, Semantic Sitemap.
Deploy: Netlify (CI/CD).
